The xmpp_xmlrpc module provides an xmlrpc transport layer to the ejabberd server.
This allows you to do server set and get commands through the xmppframework.

INSTALLATION
------------

Install and enable the xmpp xmlrpc Drupal module as you would any Drupal module.

Configure the module at Administer > XMPPFramework > XMLRPC

DEPENDENCIES
------------
Module is written purely for Drupal 6.x core.

Module depends on the XMPP Framework module

Module depends on ejabberd and mod_xmlrpc for ejabberd.
