The xmpp_api module is a pure xmpp implementation of the transport layer that 
allows you to connect to any xmpp server that supports the protocol. Module 
utilizes XEP-0133 for server administration so that certain tasks can be 
accomplished

INSTALLATION
------------

Install and enable the xmpp api Drupal module as you would any Drupal module.

Configure the module at Administer > XMPPFramework > XMPP API

DEPENDENCIES
------------
Module is written purely for Drupal 6.x core.

Module depends on the XMPP Framework module
