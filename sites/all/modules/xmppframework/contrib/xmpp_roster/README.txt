The xmpp roster module provides the ability to synchronize your xmpp server roster
with your drupal user relationships.

INSTALLATION
------------

Install and enable the xmpp roster Drupal module as you would any Drupal module.

Configure the module at Administer > XMPPFramework > Roster

DEPENDENCIES
------------
Module is written purely for Drupal 6.x core.

Module depends on the XMPP Framework module

Module depends on the User Relationships module
