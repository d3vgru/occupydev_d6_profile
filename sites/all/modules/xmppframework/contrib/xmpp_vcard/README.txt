The xmpp vcard module provides the ability to synchronize your xmpp server vcard
with your drupal content profile that is associated with the user

INSTALLATION
------------

Install and enable the xmpp vcard Drupal module as you would any Drupal module.

Configure the module at Administer > XMPPFramework > Vcard

DEPENDENCIES
------------
Module is written purely for Drupal 6.x core.

Module depends on the XMPP Framework module

Module depends on the Content Profile module

Module depends on the User Relationship Field Permissions module
