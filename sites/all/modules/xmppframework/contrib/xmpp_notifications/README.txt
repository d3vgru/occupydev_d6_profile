The xmpp notifications module provides a hook into the notifications framework so you can 
notify users of xmpp events that occur.

Currently supported events are when a user comes on and offline on the xmpp server.

INSTALLATION
------------

Install and enable the XMPP Notifications Drupal module as you would any Drupal module.

Configure the module at Administer > Notifications > XMPP

DEPENDENCIES
------------
Module is written purely for Drupal 6.x core.

Module depends on the Notifications module
